# Forum (WIP)

This is a learning project for the Hono library with Typescript, Prisma and a PostgreSQL database

## Usage

##### Setup environment

On Windows
```powershell
copy .env.template .env
```

On Linux
```
cp .env.template .env
```

##### Setup database and migrations

```
npm run init:db
```

##### Run the application

```
npm install

npm run dev
```
